# AGENTS.md

## Project Scope

These instructions apply to the entire `milestonedevelopment-site` project.

This site is the public brochure website for **Milestone Development**. Keep every change aligned with the premium construction and property development direction: modern, architectural, trustworthy, polished, mobile-friendly, and conversion-focused.

The website should feel suitable for residential construction, commercial construction, renovations, industrial projects, project management, and home-and-land package clients.

## Current Stack

* Vite
* Vanilla HTML
* CSS
* JavaScript
* SVG logo assets
* Vercel for production deployment

Do not introduce React, Tailwind, Framer Motion, or large UI libraries unless the user explicitly asks for a rebuild.

Keep the site lightweight, fast, and easy to deploy.

## Component And Code Patterns

Keep the site simple and maintainable.

Rules:

* Prefer clean semantic HTML.
* Use existing CSS variables before adding new colours.
* Keep repeated section patterns consistent.
* Use inline SVG icons or existing SVG assets.
* Keep JavaScript minimal.
* Do not add dependencies for simple animations or interactions.
* Keep mobile responsiveness explicit.
* Test text wrapping on mobile.
* Make sure buttons and forms are usable on touch devices.

For animations:

* Use subtle CSS transitions or IntersectionObserver-based reveal effects.
* Keep animations smooth and purposeful.
* Do not add heavy animation libraries.
* Avoid layout shift.

## SEO Requirements

Every production change should preserve or improve SEO.

Check:

* `<title>` is relevant.
* Meta description is strong and not duplicated.
* Canonical URL points to the production domain.
* Open Graph tags are present.
* Twitter card tags are present.
* LocalBusiness / construction business JSON-LD is valid.
* Headings follow a clean hierarchy.
* Images or visual placeholders have useful alt text where relevant.
* Phone number is visible and clickable.
* Contact section is easy to find.

Recommended title:

`Milestone Development | Residential & Commercial Construction`

Recommended meta description:

`Milestone Development provides residential, commercial, industrial and renovation construction services with experienced project management and trusted workmanship.`

## Accessibility Requirements

Preserve:

* Semantic sections
* Proper heading order
* Visible focus states
* Keyboard-accessible mobile menu
* Clear link/button labels
* Sufficient colour contrast
* Form labels or accessible names
* No hidden interactive elements that trap focus
* No text overflow on mobile

For mobile menu work:

* Toggle button must have `aria-expanded`.
* Menu should be reachable by keyboard.
* Menu should close cleanly after navigation.

## Performance Requirements

Keep the site fast.

Rules:

* No heavy libraries.
* No unnecessary external fonts/scripts.
* Optimise SVGs.
* Avoid large background images unless compressed.
* Avoid layout shift.
* Do not block rendering with unnecessary JavaScript.
* Maintain strong Lighthouse performance.

Before deploying, run a production build and check for console errors.
